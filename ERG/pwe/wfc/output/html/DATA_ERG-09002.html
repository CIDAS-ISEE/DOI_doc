DATASET OVERVIEW

Description(s)

Abstract

The dataset includes electric field spectrum data measured by the PWE/WFC instrument aboard the ERG (Arase) satellite, which are archived in the Common Data Format (CDF). The data files can be accessed with various data analysis software such as SPEDAS as well as with web browsers.
