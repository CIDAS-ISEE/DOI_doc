DATASET OVERVIEW

Description(s)

Abstract

The dataset includes spin-fit electric field data measured by the PWE/EFD instrument aboard the ERG (Arase) satellite, which are archived in the Common Data Format (CDF). The data files can be accessed with various data analysis software such as SPEDAS as well as with web browsers.
