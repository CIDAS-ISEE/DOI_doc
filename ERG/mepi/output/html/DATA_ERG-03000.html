DATASET OVERVIEW

Description(s)

Abstract

The dataset includes 3-D distributions in velocity space of energetic ions with mass discrimination measured by the MEP-i instrument onboard the ERG (Arase) satellite, which are archived in the Common Data Format (CDF). The data files can be accessed with various data analysis software such as SPEDAS as well as with web browsers.
