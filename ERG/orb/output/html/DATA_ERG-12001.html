DATASET OVERVIEW

Description(s)

Abstract

This dataset includes ERG orbit information that are calculated using the International Radiation Belt Environment Modeling (IRBEM) library with internal magnetic field model of IGRF-13 and external field performed by OP77Q, T89, and TS04 model, which are archived in the Common Data Format (CDF) format. The data files can be accessed with various data analysis software such as SPEDAS, and with web browsers.

Resource Type

Dataset, Level 3 ERG orbit information with external field performed by OP77Q, T89, and TS04 model
