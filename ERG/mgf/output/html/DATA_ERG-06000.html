DATASET OVERVIEW

Description(s)

Abstract

The dataset includes high-resolution (64/128/256 Hz) magnetic field data measured by the MGF instrument onboard the ERG (Arase) satellite, which are archived in the Common Data Format (CDF). The data files can be accessed with various data analysis software such as SPEDAS as well as with web browsers.
