DATASET OVERVIEW

Description(s)

Abstract

The dataset includes 2-D distributions in velocity space of energetic electrons measured by the XEP instrument onboard the ERG (Arase) satellite, which are archived in the Common Data Format (CDF). The data files can be accessed with various data analysis software such as SPEDAS as well as with web browsers.
