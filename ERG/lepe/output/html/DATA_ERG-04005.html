DATASET OVERVIEW

Description(s)

Abstract

This dataset includes pitch angle distribution (PAD) of low-energy electron fluxes measured by the fine channels of the LEPe instrument onboard the ERG (Arase) satellite. The MGF Level-2 8 sec data is utilized to calculate the pitch angle from the ambient magnetic field. The data are archived in the CDF format. The data files can be accessed with web browsers or data analysis software such as SPEDAS.
