DATASET OVERVIEW

Description(s)

Abstract

This dataset includes omni-directional flux in velocity space of low-energy electrons (FEDO) measured by the LEPe instrument onboard the ERG (Arase) satellite. The data are archived in the Common Data Format (CDF) format. The data files can be accessed with web browsers or data analysis software such as SPEDAS.
