DATASET OVERVIEW

Description(s)

Abstract

The dataset includes 100 Hz sampling optical intensity data from the EMCCD camera, which are archived in the raw format data files. The data files can be accessed with various data analysis software such as SPEDAS as well as with web browsers.

Resource Type

Dataset, 100 Hz sampling raw all-sky image data
